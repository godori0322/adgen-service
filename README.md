# 📚 프로젝트 개요

AdGen은 소상공인을 위한 **음성 기반 마케팅 콘텐츠 생성 서비스**입니다.  
사용자는 말 한마디와 제품 사진 한장이면 AI가 다음을 자동으로 생성합니다:

- 광고용 **이미지**
- 광고용 **음악**
- 이미지 + 음악을 합성한 **짧은 영상**

전문적인 디자인·음악·영상 편집 능력이 없어도  
**“오늘 홍보 콘텐츠 하나 만들어줘”** 같은 말로 광고 제작이 가능합니다.

---

# 🎯 핵심 기능

## ✔ 1. 음성 기반 요구 분석
- 사용자는 복잡한 입력 없이 **대화와 제품 사진 한장**이면 충분합니다.  
- “영상으로 만들어줘”, “포스터 하나 만들어줘” 같은 **모호한 말도 자연스럽게 이해**해 적절한 형태의 광고 콘텐츠를 제안합니다.

---

## ✔ 2. 콘텐츠 생성 모드 제공 (3가지)
사용자는 상황에 따라 아래 3가지 방식 중 원하는 출력 형태를 선택할 수 있습니다:

### **① 이미지 생성 모드**
- 원하는 분위기·스타일을 자동으로 해석해 **광고용 이미지**를 생성합니다.  
- 제품 사진을 업로드하면, 별도 편집 없이 **예쁜 배경과 제품이 돋보이는 광고 이미지**가 자동 구성됩니다.

### **② 이미지 + 음악 생성 모드**
- 완성된 광고 이미지에 더해, 분위기에 어울리는 **짧은 배경 음악**을 자동 생성합니다.  
- SNS 게시물, 매장 홍보용 음원 등으로 활용할 수 있습니다.

### **③ 이미지 + 음악 → 영상 생성 모드**
- 생성된 이미지와 음악을 자동으로 조합해 **짧은 홍보 영상**을 제작합니다.  
- 영상 편집 경험이 없어도 릴스/쇼츠 형태의 완성된 콘텐츠를 바로 다운로드할 수 있습니다.

---

## ✔ 3. Packshot 광고 생성
- 사용자가 올린 실제 제품 사진을 기반으로  
  **제품이 돋보이도록 구성된 Packshot 스타일 광고 이미지를 자동 생성**합니다.  
- 제품 촬영·편집 없이도 전문 광고 크리에이티브를 만들 수 있습니다.

---

## ✔ 4. 프롬프트 없이도 사용 가능
- 전문적인 프롬프트 지식이 필요 없습니다.  
- 사용자는 음성 또는 자연스러운 대화를 통해 원하는 분위기·콘셉트를 설명하기만 하면 됩니다.  
- 멀티턴 챗봇이 **사용자의 업종, 상황, 선호 스타일** 등을 대화 흐름 속에서 파악하고, 그 정보를 기반으로  
  **최적화된 광고 아이디어·문구·이미지 콘셉트**를 자동으로 제안합니다.
- 생성 과정에서는 버튼을 통해 **Rigid / Balanced / Creative** 모드 중 하나를 선택해  
  이미지의 창의성·변형 정도를 조절할 수 있습니다.
- 포스터에 들어갈 텍스트도 UI에서 간단히 선택할 수 있어  
  “문구 + 이미지 + 음악/영상”까지 하나의 흐름으로 만들 수 있습니다.
- 단순한 이미지 생성 도구가 아니라,  
  **사용자 맞춤형 마케팅 어시스턴트**처럼 광고 제작 전 과정을 함께 도와줍니다.


---

## ✔ 5. SNS 공유 기능
- 생성된 이미지·음원·영상은 **즉시 다운로드 가능**하며,  
- 모바일 환경에서는 **공유 패널을 통해 인스타그램·카카오톡 등으로 바로 공유**할 수 있어 활용이 쉽습니다.

# 🧠 기술 스택

### **AI 모델**
- Whisper API — 음성 인식
- GPT-4o-mini — 마케팅 문구 및 이미지·음악 프롬프트 생성
- Stable Diffusion 1.5 — 이미지 생성
- ControlNet / IP-Adapter — 제품 기반 포스터 강화
- SAM / MobileSAM — 누끼
- **MusicGen API — 광고용 음악 생성**

### **Backend**
- FastAPI  
- MinIO (이미지·음원·영상 저장)  
- MoviePy (이미지+음악 영상 합성)

### **Frontend**
- React  
- Zustand  
- PWA 지원

### **Infra**
- GCP VM (L4 GPU)  
- Nginx Reverse Proxy  
- DuckDNS + HTTPS (Certbot)

# 🌐 배포 구조

AdGen은 단일 VM에서 동작하며, Nginx가 다음과 같이 라우팅합니다:

| 리소스 | 접근 방식 | Nginx 내부 라우팅 |
|--------|-----------|-------------------|
| 백엔드(API) | https://adgen4.duckdns.org/api/... | http://localhost:8080/api/... |
| MinIO | https://adgen4.duckdns.org/media/... | http://localhost:9000/... |
| 프론트엔드 | https://adgen4.duckdns.org/ | /var/www/adgen/dist |

# 🖼 서비스 흐름

1. 사용자가 음성 입력  
2. Whisper가 텍스트 변환  
3. GPT가 이미지·음악 생성에 필요한 프롬프트 및 문구 생성  
4. 사용자 선택 모드에 따라 다음 중 하나 생성  
   - 이미지  
   - 이미지 + 음악  
   - 이미지 + 음악 → 영상  
5. MinIO에 저장 후 UI에 표시  
6. 다운로드하여 SNS 등에서 활용 가능  

(시연 영상은 추후 README에 추가 예정)
